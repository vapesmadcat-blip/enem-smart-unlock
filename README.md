# ENEM Smart Unlock 🧠📱

Um aplicativo Android inovador que transforma a tela de bloqueio em uma ferramenta de estudo ativo para o ENEM. O usuário deve responder corretamente 3 questões do ENEM para desbloquear o dispositivo.

## ✨ Características

- **50+ Questões Reais:** Banco expandido com questões de 7 disciplinas
- **Tela de Bloqueio Inteligente:** Ativa automaticamente ao desbloquear
- **Sistema de Progresso:** Acompanhamento visual com barra de progresso
- **Feedback Haptic:** Vibrações para feedback tátil
- **Tratamento de Erros:** Aplicação robusta com exceções tratadas
- **Design Dark Mode:** Interface otimizada para tela de bloqueio

## 🚀 Compilação Rápida

```bash
# Debug
./gradlew assembleDebug

# Release
./gradlew assembleRelease
```

APK gerado em: `app/build/outputs/apk/`

## 📱 Instalar

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 🎯 Como Usar

1. Abra o app
2. Toque em "INICIAR DESAFIO"
3. Responda 3 questões corretamente
4. Dispositivo será desbloqueado
